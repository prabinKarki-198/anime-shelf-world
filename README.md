# Manga Verse

A manga browsing and reading application built with React, TypeScript, and Vite.

## Features

- Browse popular manga
- Search for manga by title
- View manga details and chapters
- Read chapters with an image viewer
- Recommendations based on similar tags

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- React Query (Tanstack Query)
- React Router
- Shadcn/ui components
- MangaDex API

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## API

This project uses the [MangaDex API](https://api.mangadex.org) to fetch manga data.
